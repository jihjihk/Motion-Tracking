How to Start

Time to flap and turn up the volume

When you first launch the page and see the camera screen, flap your hand to activate each object that is floating on the screen. Each object will bring up different songs and virtual worlds.

Every time you flap and get closer to fully activating each world, the volume of the background music will get louder.

If the browser does not ask for permission to allow camera access or load any screen below this text, please try typing https:// in front of the url and reload.
